
Exercícios de Funções em C (Nível Médio)

1. Função para verificar se um número é primo:
   Escreva uma função chamada 'eh_primo' que receba um número inteiro e retorne 1 se o número for primo e 0 caso contrário.
   Teste a função no programa principal imprimindo todos os números primos entre 1 e 100.

2. Função para calcular a potência:
   Implemente uma função 'potencia' que receba dois parâmetros inteiros 'base' e 'expoente' e retorne o valor da base elevada ao expoente. Não use a função pow da biblioteca. Teste a função no programa principal.

3. Função que calcula a média de um array:
   Escreva uma função chamada 'media' que receba um array de inteiros e o tamanho do array. A função deve calcular e retornar a média dos valores. No programa principal, leia 5 números do usuário, armazene-os em um array e passe o array para a função calcular a média.

4. Função recursiva para calcular o fatorial:
   Crie uma função recursiva chamada 'fatorial' que receba um número inteiro e retorne o fatorial desse número. Teste a função chamando-a no programa principal para calcular o fatorial de um número fornecido pelo usuário.

5. Função que inverte uma string:
   Implemente uma função 'inverte_string' que receba uma string (vetor de caracteres) e retorne a string invertida. Teste a função no programa principal lendo uma string do usuário e imprimindo a versão invertida.

6. Função para contar dígitos de um número:
   Escreva uma função chamada 'conta_digitos' que receba um número inteiro e retorne a quantidade de dígitos que ele possui. Por exemplo, se o número for 12345, a função deve retornar 5.

7. Função para calcular o Máximo Divisor Comum (MDC):
   Crie uma função chamada 'mdc' que receba dois números inteiros e retorne o maior número que divide ambos. Use o algoritmo de Euclides para encontrar o MDC. Teste a função no programa principal.

8. Função para verificar palíndromo:
   Escreva uma função 'eh_palindromo' que receba uma string e retorne 1 se a string for um palíndromo e 0 caso contrário. Uma string é palíndromo se ela pode ser lida da mesma forma de trás para frente, ignorando espaços e diferenças de maiúsculas/minúsculas.

9. Função para converter temperatura:
   Implemente uma função 'converte_temperatura' que receba dois parâmetros: a temperatura em Celsius e um caractere indicando se ela deve ser convertida para Fahrenheit ('F') ou Kelvin ('K'). A função deve retornar a temperatura convertida. Teste no programa principal.

10. Função para contar vogais em uma string:
   Escreva uma função 'conta_vogais' que receba uma string e retorne o número de vogais (a, e, i, o, u) presentes na string. Teste a função no programa principal.
