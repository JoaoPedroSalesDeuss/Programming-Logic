1. Variáveis e Tipos de Dados
   1. Declare e inicialize variáveis de diferentes tipos (int, float, char) e imprima seus valores.
   2. Crie um programa que converta graus Celsius para Fahrenheit.
   3. Escreva um programa que calcule a área de um círculo dada sua razão.

2. Estruturas de Controle
   4. Crie um programa que leia um número e informe se ele é par ou ímpar.
   5. Implemente um programa que leia três números e imprima o maior deles.
   6. Escreva um programa que some todos os números inteiros de 1 a N, onde N é fornecido pelo usuário.
   7. Crie um programa que imprima os números de 1 a 100, mas substitua múltiplos de 3 por 'Fizz' e múltiplos de 5 por 'Buzz'.

3. Estruturas de Repetição
   8. Escreva um programa que imprime a tabela de multiplicação de um número fornecido pelo usuário.
   9. Crie um programa que calcule a soma de todos os dígitos de um número inteiro fornecido pelo usuário.
   10. Faça um programa que imprima os números de Fibonacci até o N-ésimo termo.

4. Funções
   11. Crie uma função que receba dois números e retorne o maior deles.
   12. Escreva uma função que calcule e retorne o fatorial de um número inteiro.
   13. Implemente uma função que receba um array de inteiros e seu tamanho, e retorne a soma dos elementos.
   14. Crie uma função que determine se um número é primo.

5. Arrays
   15. Escreva um programa que armazene 10 números em um array e calcule a média.
   16. Crie um programa que inverta os elementos de um array.
   17. Faça um programa que leia um array de inteiros e encontre o menor e o maior valor.
   18. Implemente um programa que ordene um array de inteiros em ordem crescente (algoritmo de bolha).

6. Strings
   19. Crie um programa que leia uma string e imprima seu comprimento.
   20. Implemente uma função que conte o número de vogais em uma string.
   21. Escreva um programa que verifique se duas strings são palíndromas.
   22. Crie um programa que converta uma string para maiúsculas.

7. Ponteiros
   23. Escreva um programa que demonstre o uso de ponteiros com variáveis inteiras.
   24. Crie uma função que troca o valor de duas variáveis usando ponteiros.
   25. Implemente uma função que recebe um array de inteiros e seu tamanho, e usa ponteiros para calcular a soma dos elementos.

8. Estruturas
   26. Defina uma estrutura para armazenar informações de um aluno (nome, idade, nota) e escreva um programa que leia essas informações.
   27. Crie um programa que armazene e imprima dados de vários alunos usando arrays de estruturas.

9. Manipulação de Arquivos
   28. Escreva um programa que cria um arquivo e grava 10 números inteiros nele.
   29. Crie um programa que lê os números de um arquivo e os imprime na tela.
   30. Faça um programa que lê números de um arquivo e calcula a média deles.

10. Recursão
   31. Escreva uma função recursiva que calcula o fatorial de um número.
   32. Implemente uma função recursiva que determina se um número é primo.
   33. Crie uma função recursiva que imprime os números de Fibonacci até o N-ésimo termo.

11. Exercícios de Nível Avançado
   34. Escreva um programa que usa ponteiros duplos para manipular uma matriz.
   35. Crie uma função que ordena uma lista ligada de inteiros.
   36. Implemente um programa que resolve o problema das Torres de Hanói usando recursão.

12. Projetos Finais
   37. Crie um sistema de gerenciamento de alunos com CRUD (Criar, Ler, Atualizar, Deletar) usando estruturas.
   38. Implemente uma calculadora simples que suporte as operações básicas (soma, subtração, multiplicação, divisão).
   39. Desenvolva um jogo da velha em C, onde o usuário pode jogar contra o computador.
   40. Crie um programa que simula um caixa eletrônico com operações de saque, depósito e saldo.

13. Desafios de Lógica
   41. Implemente um programa que verifica se uma palavra é um anagrama de outra.
   42. Crie um programa que encontra o maior palíndromo que pode ser obtido a partir do produto de dois números de 3 dígitos.
   43. Faça um programa que encontra todos os números primos até um número N.
   44. Implemente uma função que calcula a soma de todos os dígitos de um número, usando recursão.

14. Manipulação Avançada de Strings
   45. Crie um programa que conta a frequência de cada letra em uma string.
   46. Implemente uma função que remove espaços em branco de uma string.
   47. Faça um programa que substitui todas as ocorrências de uma palavra em uma string por outra palavra.
   48. Crie um programa que verifica se uma string é um palíndromo, desconsiderando espaços e pontuação.

15. Trabalhando com Erros
   49. Implemente um programa que trata erros de leitura de arquivo e exibe mensagens apropriadas.
   50. Crie uma função que verifica se a divisão por zero ocorreu e exibe uma mensagem de erro caso aconteça.
